# ☁️ Deploy on Render

## 1️⃣ Connect Repository

Create a new Render Web Service and connect your GitHub repository.

---

## 2️⃣ Configure Secrets

Navigate to:

```bash
Render Dashboard → Environment
```

Add:
- API_KEY
- JWT_SECRET
- MY_PASSWORD
- PORT

---

## 3️⃣ Build Settings

### Build Command

```bash
npm install
```

### Start Command

```bash
node server.js
```

---

## 4️⃣ Auto Deployment

Every GitHub push triggers automatic deployment updates.

---
